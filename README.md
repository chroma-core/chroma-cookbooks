# Chroma Cookbooks

The Chroma Cookbooks provide guides and complete code examples for building AI applications powered by [Chroma](https://trychroma.com). Comprehensive guides for these projects can be found on Chroma's [docs](https://docs.trychroma.com/guides/build/building-with-ai).

## Prerequisites

Many of the projects in this repo require a Chroma Cloud account. You can [signup](https://trychroma.com/signup) and get free credits to interact with the cookbook projects and start building your own.
